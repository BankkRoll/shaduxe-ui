## Quick Start

<Steps>
  <Step>
    Install Dependencies
    
    ```bash
    npx shadcn@latest init
    ```
  </Step>

  <Step>
    Add Components 
    
    ```bash
    npx shadcn@latest add button
    ```
  </Step>

  <Step>
    Use in Your Project
    
    ```tsx showLineNumbers
    import { Button } from "@/components/ui/button"

    export default function Page() {
      return <Button>Click me</Button>
    }
    ```
  </Step>
</Steps>

## FAQ

<Accordion type="multiple" className="w-full">
  <AccordionItem value="faq-1">
    <AccordionTrigger>
      How is shaduxe/ui different from shadcn/ui?
    </AccordionTrigger>
    <AccordionContent>
      While shadcn/ui provides foundational components, shaduxe/ui offers pre-built variations with advanced features like:
      - Modern animations and transitions
      - Additional style variants
      - Complex component combinations
      - Ready-to-use templates
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-2">
    <AccordionTrigger>
      Do I need to install shaduxe/ui separately?
    </AccordionTrigger>
    <AccordionContent>
      No, shaduxe/ui components are direct extensions of existing shadcn/ui
      components. Just copy the code you need and customize it.
    </AccordionContent>
  </AccordionItem>

  <AccordionItem value="faq-3">
    <AccordionTrigger>
      What frameworks are supported?
    </AccordionTrigger>
    <AccordionContent>
      shaduxe/ui supports all frameworks compatible with shadcn/ui:
      - Next.js
      - Remix
      - Astro
      - Gatsby
      And any other React-based framework.
    </AccordionContent>
  </AccordionItem>
</Accordion>
