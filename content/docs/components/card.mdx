## Usage

```tsx {1-6,10-19} showLineNumbers
import { 
  Card, 
  CardContent, 
  CardDescription, 
  CardFooter, 
  CardHeader, 
  CardTitle 
} from "@/components/ui/card";

export default function Home() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card Description</CardDescription>
      </CardHeader>
      <CardContent>Main content goes here</CardContent>
      <CardFooter>Footer content goes here</CardFooter>
    </Card>
  );
}
```

## Props

| Prop      | Type                                                                      | Default   | Description                |
| --------- | ------------------------------------------------------------------------- | --------- | -------------------------- |
| variant   | `default` \| `flat` \| `elevated` \| `ghost` \| `outline` \| `opaque` \| `gradient` | `default` | Style variant of the card  |
| size      | `sm` \| `md` \| `lg` \| `xl`                                              | `md`      | Size of the card padding   |
| radius    | `default` \| `sm` \| `md` \| `lg` \| `full` \| `none`                     | `default` | Border radius of the card  |
| className | `string`                                                                  | -         | Additional CSS classes     |

### Supported HTML Attributes

All Card components use standard HTML div elements, so they inherit and support **all standard HTML div attributes**, including:

- `id`
- `onClick`
- `role`
- `aria-*` attributes
- `data-*` attributes
- Event handlers like `onMouseEnter`, `onFocus`, etc.

The Card components ensure proper accessibility and semantic HTML.
