## Props

| Prop         | Type                                        | Default   | Description                       |
| ------------ | ------------------------------------------- | --------- | --------------------------------- |
| variant      | `default` \| `rounded` \| `square` \| `ios` | `default` | Style variant of the switch       |
| size         | `icon` \| `xs` \| `sm` \| `md` \| `lg`      | `md`      | Size of the switch                |
| thumbVariant | `default` \| `rounded` \| `square` \| `ios` | -         | Style variant of the switch thumb |
| showLabels   | `boolean`                                   | `false`   | Show text labels                  |
| labelOn      | `string`                                    | `"On"`    | Text for the "on" label           |
| labelOff     | `string`                                    | `"Off"`   | Text for the "off" label          |
| showIcons    | `boolean`                                   | `false`   | Show icons in the switch thumb    |
| className    | `string`                                    | -         | Additional CSS classes            |

### Supported HTML Attributes

Because the Switch uses Radix UI's `Switch` component, it inherits and supports **all standard HTML input attributes**, including:

- `checked`
- `defaultChecked`
- `required`
- `name`
- `value`
- `onCheckedChange`
- `disabled`
- `aria-*` attributes
- `data-*` attributes
- Event handlers like `onFocus`, `onBlur`, etc.

The Switch component uses Radix UI's primitives, ensuring proper accessibility, keyboard navigation and semantic HTML.
