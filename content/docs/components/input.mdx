## Props

| Prop          | Type                                    | Description                                | Default   |
|---------------|-----------------------------------------|--------------------------------------------|-----------|
| variant       | `default` \| `underline` \| `pill`    | Style variant of the input                  | `default` |
| inputSize     | `xs` \| `sm` \| `md` \| `lg` \| `xl` | Size of the input                           | `md`      |
| type          | `text` \| `email` \| `password` \| etc. | Type of the input field                     | `text`    |
| className     | `string`                                | Additional CSS classes                     | -         |
| Icon          | `React.ElementType`                     | Icon component to be displayed             | -         |
| iconPlacement | `"left"` \| `"right"`                   | Placement of the icon                       | `"left"`  |

### Supported HTML Attributes

Because the Button uses Radix UI, it inherits and supports **all standard HTML input attributes**, including:

- `placeholder`
- `disabled`
- `required`
- `readOnly`
- `value`
- `onChange`
- `onFocus`
- `onBlur`
- `aria-*` attributes
- `data-*` attributes

The Input component uses Radix UI's primitives, ensuring proper accessibility, keyboard navigation and semantic HTML.
